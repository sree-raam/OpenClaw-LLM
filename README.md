# OpenClaw-LLM

## Latency-Aware ROS 2 Framework for LLM-Based Task Planning and Autonomous Navigation

This project implements a latency-aware intelligent robotic framework for natural language task planning, autonomous navigation, safety supervision, and performance benchmarking in ROS 2 mobile robots. The framework integrates lightweight Large Language Models (LLMs) deployed through Ollama to generate navigation missions from human instructions and execute them within a waypoint-based navigation pipeline.

The project was developed as a Final Year Project (FYP) at Universiti Teknologi Malaysia (UTM), Malaysia-Japan International Institute of Technology (MJIIT), and evaluated on NVIDIA Jetson Orin Nano using TurtleBot3 Burger in Gazebo simulation environments.

---

# Key Features

### 1. LLM-Based Task Planning

Supports natural language commands for autonomous mission generation.

Examples:

```text
Inspect Cylinder 1
Inspect Cylinders 2, 5 and 8
Inspect nearest cylinder
Inspect all cylinders
Return Home
```

Supported models:

* Gemma2-2B
* Qwen2.5-3B
* DeepSeek-R1-1.5B


---

### 2. Autonomous Navigation

Features:

* ROS 2 Humble integration
* Waypoint-based navigation
* Gazebo Classic simulation
* Mission execution monitoring
* Autonomous Home return
* Sequential inspection tasks


---

### 3. Safety Supervisor

Safety mechanisms include:

* Collision detection
* Mission timeout monitoring
* Duck-walk recovery
* Navigation retry mechanism
* Emergency stop handling
* Autonomous mission interruption recovery


---

### 4. Latency-Aware Benchmark Evaluation

Performance metrics collected include:

* LLM inference latency
* Navigation completion time
* Mission success rate
* CPU utilization
* Memory consumption
* Path efficiency
* Recovery effectiveness
* System stability

Input:

---

Examples:
